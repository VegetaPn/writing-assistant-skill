## 2. 双轴分布规律

> 数据来源：`references/by-element/titles/dontbesilent-collection.md` 统计部分

### 结构偏好

| 结构 | 占比 | 特点 |
|------|------|------|
| **陈述/宣言** | **47.1%** | 绝对主力，直接断言不解释 |
| 叙事/事件钩子 | 30.1% | 第二主力，个人经历开头 |
| 提问 | 9.8% | 克制使用，不过度依赖 |
| 等式/公式 | 8.5% | 精准定义，"A = B" 结构 |
| 对比/并列 | 4.6% | 少用但效果极强 |

### 机制偏好

| 机制 | 出现次数 | 占比 | 特点 |
|------|---------|------|------|
| **反常识** | **73** | **33.2%** | 核心武器，几乎每两条就有一条 |
| 信息缺口 | 43 | 19.5% | 叙事类标题的标配 |
| 社会证明 | 31 | 14.1% | 数据背书，但不空有数据 |
| 情绪共鸣 | 22 | 10.0% | 焦虑和愤怒为主 |
| 根因揭示 | 21 | 9.5% | 经常与反常识叠加 |
| 时间压缩 | 18 | 8.2% | AI/效率类内容专用 |
| 命名 | 15 | 6.8% | 最高级技法，用于概念创造 |

### 核心规律

1. **反常识是绝对主力**：33.2% 的机制出现频率，说明 dontbesilent 的标题策略本质上就是"说和大家相反的话"
2. **叙事+社会证明+信息缺口 是黄金三角**：几乎所有叙事类标题都同时使用这三者
3. **命名是最高级技法**：出现频率最低（6.8%），但代表作含金量最高（"习得性无助""打工思维""心力不足"）
4. **对比/并列虽然只有 4.6%，但每条都是精品**：如 "像素级复制，年入百万。搬运抄袭，颗粒无收"

## 3. 语言特征

### 3.1 字数分布

- **主力区间**: 15-25 字（符合小红书/抖音标题要求）
- **极短标题**: "分享创业干货"（5字）— 罕见
- **较长标题**: "3 分钟教会你从零到一找到自己的生意❗️公开我的表格工作法"（30字）— 偶尔突破

### 3.2 标点使用

- **冒号（：）** 是标志性标点。大量使用冒号分割前后半句，形成"设局：颠覆"结构
  - "一个人赚不到钱的核心原因就是：上班上多了"
  - "卖了几百万货我发现：人买的不是货，是幻想"
- **竖线（｜）** 用于补充信息或标注系列
  - "总想把产品做得太有用，是在耽误你赚钱｜商业反直觉"
- **Emoji** 偶尔使用，不超过 1-2 个
  - 😂 😭 🔥 ❗️

### 3.3 人称

- 大量使用**第一人称"我"**，增加真实感
  - "我用""我发现""我设计""我的方法"
- 偶尔使用**"你"**直接对话
  - "你真正想要的是""你展示了你的工具性"
- **绝不使用"您"** — 无距离感

### 3.4 数字使用

- **高频使用具体数字**：70 万、40 万、8 千人、100 张图、¥17、¥3.9
- **偏好非整数**：¥17 而不是 ¥20，强化真实感
- **百分比用法**：99%、100 倍 — 制造极端感
- **时间数字**：3 分钟、5 分钟、15s、24 小时、8 个月、3 年、30 年

### 3.5 口语化程度

- **高度口语化**：像朋友聊天
- 使用粗口/俚语："耍流氓""精神鸦片""遮羞布""烂好人""坑死自己""扯淡"
- 使用网络梗："东哥的兄弟""爆哭""喂饭版"
- **不学术化**：不用"赋能""底层逻辑"等词（"底层逻辑"偶有使用但通常配合反讽或具体场景）

## 4. 核心句式模板

### 模板 1: 冒号分割法 — "[设局]：[颠覆/揭示]"

前半句设局（经历/数据/观察），冒号后给出反常识结论。

**适用结构**: 陈述/宣言、叙事/事件钩子
**适用机制**: 根因揭示 + 反常识

| 例子 | 分析 |
|------|------|
| 一个人赚不到钱的核心原因就是：上班上多了 | "核心原因"设局 → "上班上多了"颠覆 |
| 卖了几百万货我发现：人买的不是货，是幻想 | 经验牌设局 → 商业本质颠覆 |
| 扎心：让你爽的品牌标语，背后全是商业效率 | 情绪词设局 → 揭示商业逻辑 |
| 连线几百人之后我发现：不挣钱的业务，都长一个样 | 数据设局 → 规律揭示 |

---

### 模板 2: 经验牌开局 — "[数据/时间+我]...[发现/之后]..."

以第一人称+具体数据开头，先建立信任再抛观点或留悬念。

**适用结构**: 叙事/事件钩子
**适用机制**: 社会证明 + 信息缺口

| 例子 | 分析 |
|------|------|
| 去年涨粉 70 万后，我所有方法压缩成了一句话 | 70万背书 + "一句话"极致悬念 |
| 年入百万之后，发现最重要的赚钱技能是定价 | 收入背书 + 单一技能揭示 |
| 40 万粉丝后，我发现 99% 的人不会看数据 | 粉丝数背书 + 反常识发现 |
| 连续 8 个月每天涨粉 1 千，我是如何做到的 | 持续性数据 + 方法悬念 |

---

### 模板 3: 反定义法 — "X 不是 Y，是/而是 Z"

否定一个概念的常规定义，给出新定义。

**适用结构**: 等式/公式
**适用机制**: 反常识

| 例子 | 分析 |
|------|------|
| AI 不是新范式，是 1960s 分工模式的低成本实现 | 否定"新范式"，降维到历史模式 |
| AI 不是用来赚钱的，是用来不被淘汰的 | 否定工具价值，强调生存价值 |
| 99% 的私域流量都不是资产，而是负债 | 否定"资产"定义，反转为负债 |

---

### 模板 4: 等式命名法 — "A = B"

直接用等号给一个模糊概念赋予精确定义。

**适用结构**: 等式/公式
**适用机制**: 命名 + 反常识

| 例子 | 分析 |
|------|------|
| 打工思维 = 对任何事都一头雾水的大脑 | 给"打工思维"一个具体画面 |
| 内耗 = 心理资本匮乏 | 把感受转化为资源概念 |
| 不相信自己能赚到钱 = 习得性无助 | 用心理学概念精准命名 |

---

### 模板 5: 对仗反差 — "[正面短句]。[反面短句]"

前后两个短句，结构对称，含义反转。

**适用结构**: 对比/并列
**适用机制**: 反常识

| 例子 | 分析 |
|------|------|
| 像素级复制，年入百万。搬运抄袭，颗粒无收 | 看似相同实则天差地别 |
| 赚钱简单，成为想赚钱的人，很难 | 同一件事的两面反转 |
| 赔钱是能力，赚钱是时机 | 颠覆"赔钱=失败"的认知 |
| 赚第一桶金用了 1 天，破除执念花了 3 年 | 时间维度的极端对比 |

---

### 模板 6: "越 X 越 Y" 悖论

用"越...越..."结构制造违反直觉的因果关系。

**适用结构**: 陈述/宣言
**适用机制**: 反常识

| 例子 | 分析 |
|------|------|
| 越研究人群画像，越赚不到钱 | 否定"精准"策略 |
| 做 1 v 1 咨询，越接单越穷，越努力越穷 | 否定"努力=回报" |
| 越有钱越艰难，过亿的老板们做短视频更痛苦 | 否定"有钱=容易" |

---

### 模板 7: "所有/99%/从来不" 绝对化

用绝对化词语制造极端断言。

**适用结构**: 陈述/宣言
**适用机制**: 反常识 + 情绪共鸣

| 例子 | 分析 |
|------|------|
| 所有教你「先做内容再变现」的人都是耍流氓 | "所有"+"耍流氓"双重极端 |
| 99% 的知识付费业务，第一天就注定赚不了钱 | "99%"+"第一天"+"注定"三重 |
| 能赚钱的知识付费从来不卖虚拟资料，卖情绪 | "从来不"绝对否定 |

---

### 模板 8: 概念命名 — 给模糊感受起名

用一个精确的词或短语命名读者心中模糊的感受。

**适用结构**: 陈述/宣言、等式/公式
**适用机制**: 命名

| 例子 | 命名的概念 | 命名了什么 |
|------|----------|-----------|
| 副业思维锁死了一个人的财富上限 | "副业思维" | 思维模式的局限性 |
| 月薪是人生最大陷阱，稳定是穷人的精神鸦片 | "精神鸦片" | 对稳定的依赖本质 |
| 总是被白嫖，是因为你展示了你的工具性 | "工具性" | 被利用的原因 |
| 逆商决定财富上限 | "逆商" | 抗挫折能力的商业价值 |
| 恢复主体性，是一个人开始赚钱的前提 | "主体性" | 自我驱动力 |

## 5. 标题哲学

### 5.1 核心理念：真钩子 vs 假钩子

dontbesilent 在文章中反复强调：好的钩子不是欺骗，是建立信任的机会。假钩子骗点击但内容不兑现，真钩子承诺并兑现。

> 详见 `references/techniques/hooks/4-factor-scoring.md`

### 5.2 四因子评分体系

标题/钩子的强度可以用四个维度评估：
- **破坏预测** — 打断大脑自动预测（对应标题机制：反常识、根因揭示）
- **奖励** — 制造多巴胺预期（对应：信息缺口、时间压缩、社会证明）
- **损失厌恶** — 害怕错过（对应：情绪共鸣）
- **命名** — 给感受赋予名字（对应：命名）

3+ 因素得分 ≥ 8/10 → 几乎必然爆款。

> 详见 `references/techniques/hooks/4-factor-scoring.md`

### 5.3 钩子层级体系

钩子按受众覆盖面分 5 级：万能 → 泛领域 → 中领域 → 窄领域 → 极窄。dontbesilent 主要在 Level 3（中领域，商业认知）和 Level 4（窄领域，具体策略）之间创作。

> 详见 `references/techniques/hooks/hook-hierarchy.md`

## 6. 可复用标题技巧总结

| 技巧名称 | 适用结构 | 适用机制 | 核心操作 |
|---------|---------|---------|---------|
| 冒号分割法 | 陈述、叙事 | 根因揭示 + 反常识 | 前半句设局，冒号后颠覆 |
| 经验牌开局 | 叙事 | 社会证明 + 信息缺口 | "我 + 数据 + 发现/之后" |
| 反定义法 | 等式 | 反常识 | "X 不是 Y，是 Z" |
| 等式命名法 | 等式 | 命名 + 反常识 | "A = B"（给概念新定义） |
| 对仗反差 | 对比 | 反常识 | 两个对称短句，含义反转 |
| "越X越Y"悖论 | 陈述 | 反常识 | 违反直觉的因果关系 |
| 绝对化断言 | 陈述 | 反常识 + 情绪共鸣 | "所有/99%/从来不" |
| 概念命名 | 陈述、等式 | 命名 | 给模糊感受一个精确名字 |

## 7. 使用指南

当 `skills/title-generator.md` 的 Step 3 引用本档案时：

1. **如果目标平台是小红书/抖音**，优先参考 dontbesilent 的句式模板
2. **核心公式：反常识 + 具体数字 + 第一人称经验**
3. **标题长度控制在 15-25 字**
4. **优先使用陈述/宣言（47%）和叙事/事件钩子（30%）结构**
5. **反常识是第一机制选择**（33.2% 出现率），但注意搭配其他机制增加维度
6. **命名技法慎用但要用好** — 如果能给读者的痛点找到一个精准的名字，标题力量会翻倍
7. **避免**：过度学术化、缺少具体画面、没有情绪触发点、用"您"
