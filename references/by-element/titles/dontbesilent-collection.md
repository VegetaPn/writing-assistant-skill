## 分类轴说明

**结构（5 种）：**
- **陈述/宣言** — 直接断言，大胆宣称
- **等式/公式** — A = B 或 A 不是 B 而是 C
- **叙事/事件钩子** — 个人经历或具体事件开场
- **对比/并列** — 两个事物并置，制造反差
- **提问** — 问句形式

**机制（7 种，标注底层 Factor）：**
- **反常识**〔破坏预测〕— 挑战读者的默认假设
- **信息缺口**〔奖励〕— 说了一半，扣住另一半
- **根因揭示**〔破坏预测〕— "真正的原因是..."
- **社会证明**〔奖励〕— 个人成就或数据背书
- **时间压缩**〔奖励〕— 短时间 → 高回报感
- **情绪共鸣**〔损失厌恶〕— 直接击中读者情绪
- **命名**〔命名〕— 给模糊感受一个精确名字
