## Method #1: 四因子钩子评分系统 (4-Factor Hook Scoring)

**Core Framework（核心框架）：**

```
┌─────────────────────────────────────────────────────────┐
│                   钩子评分公式                            │
│                                                         │
│   Factor 1: 破坏预测 ──── 打断大脑自动预测               │
│   Factor 2: 奖励承诺 ──── 制造多巴胺期待                 │
│   Factor 3: 损失厌恶 ──── 制造害怕错过的感觉             │
│   Factor 4: 命名 ──────── 给模糊感受赋予具体名称          │
│                                                         │
│   规则: 4 项中有 3 项 ≥ 8/10 → 几乎必定爆款              │
└─────────────────────────────────────────────────────────┘
```

[详细文档](4-factor-scoring.md)

**Source（来源）：** dontbesilent - 《做短视频的本质是黑掉人的认知系统》 (小红书/抖音)

**Psychology Mechanism（心理学机制）：** 人脑本质上是一台预测机器，钩子通过攻击预测系统、多巴胺系统、损失厌恶心理和自我认知的模糊地带来劫持注意力。

**Why It Works（为什么有效）：** 提供了一个可量化的评估框架，把"有没有吸引力"这个模糊问题转化为四个维度的打分，可以用 AI 辅助评估，在发布前预测流量层级。

**Practice Guide（实践指南）：** 把公式截图发给 AI，附上短视频开头文案，让 AI 对四项各打 0-10 分。3 项 8 分以上即为强钩子。

## Method #2: 五级钩子层级体系 (5-Level Hook Hierarchy)

**Core Framework（核心框架）：**

```
┌─────────────────────────────────────────────────────────┐
│                   钩子层级金字塔                          │
│                                                         │
│  Level 1 万能层 ─── 超广覆盖，后面接任何话题              │
│  Level 2 泛领域 ─── 个人成长/认知类，覆盖广               │
│  Level 3 中领域 ─── 中等细分（如商业认知），平衡覆盖与转化  │
│  Level 4 窄领域 ─── 细分哲学洞察，转化高但覆盖窄           │
│  Level 5 极窄 ──── 超具体技巧/承诺，风险最高               │
│                                                         │
│  从上到下: 覆盖面递减，转化率递增                          │
│  迁移规则: 只从粉丝量 5-10 倍的博主处借鉴钩子              │
└─────────────────────────────────────────────────────────┘
```

[详细文档](hook-hierarchy.md)

**Source（来源）：**
- dontbesilent - 《连续 8 个月每天涨粉 1 千》 (小红书/抖音)
- dontbesilent - 《一条视频，播放千万，涨粉 6 万》 (小红书/抖音)
- dontbesilent - 《每条抖音视频稳定涨粉 2000+》 (小红书/抖音)
- dontbesilent - 《如何只靠原创涨粉 20 万》 (小红书/抖音)

**Psychology Mechanism（心理学机制）：** 钩子的覆盖面与精准度是一对天然矛盾。越万能的钩子触达越广但筛选力越弱，越细分的钩子转化越强但受众越窄。创作者需要根据自己的账号阶段选择合适的层级。

**Why It Works（为什么有效）：** 把"抄什么样的钩子"从盲目行为变成了有层级意识的策略选择，同时结合跨平台验证流程（Twitter验证 → 小红书找标题 → 抖音找钩子），系统性地提升钩子命中率。

**Practice Guide（实践指南）：** 先判断自己在哪个层级创作，再从粉丝量 5-10 倍的同类博主处寻找可迁移的钩子结构。用跨平台流程验证选题→发现标题→匹配钩子。

## 如何添加新的方法论

1. **从文章/分享中识别方法论：**
   - 是否包含一个关于钩子/开头的核心框架？
   - 是否解释了"为什么这种开头有效"的原理？
   - 是否有数据支撑（播放量、点赞、完播率等）？

2. **提取并结构化：**
   - 按照上面的格式模板填写
   - 确保包含心理学机制的解释
   - 提供可操作的实践指南和数据案例

3. **关联到写作流程：**
   - 这个方法论应该在 Step 4（Element-Level Reference）阶段使用
   - 和 `by-element/hooks/` 中的案例配合使用
